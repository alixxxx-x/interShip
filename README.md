# Final Project - Internship Management Platform

A full-stack web application to streamline internship management and connect students with companies.  
The platform provides dedicated spaces for students, companies, and university administrators, handling applications, validation, and document generation efficiently.

## Features

### Student Space
- Secure login and registration.
- Digital CV with technical skills and portfolio links.
- Search and apply to internships with filters (location, skills, type).
- Track application status.

### Company Space
- Company profile management (logo, description, location).
- Create, update, and delete internship offers.
- Track student applications and accept or reject candidates.

### Administration Space
- Validate internships approved by companies.
- Automatic generation of internship agreements and certificates (PDF).
- Dashboard for placement statistics.

## Tech Stack
- **Frontend:** ReactJS  
- **Backend:** Django  
- **Database:** PostgreSQL  
- **Authentication:** JWT-based login  

## Skills Gained
- Full-stack web development (React + Django)  
- Database design and management (PostgreSQL)  
- File handling and PDF generation  
- Deployment of web applications
